Includes selectable fonts to WordPress Theme, can remove all Google Fonts (and Google APIs) code from Theme.
 
== Description ==
 
This Plugin includes selectable fonts to your website. Besides this, it can remove all Google Fonts and Google APIs (non GDPR compliant third party code) from your website.
 
== Installation ==
 
1. Upload the plugin folder to your /wp-content/plugins/ folder.
2. Go to the **Plugins** page and activate the plugin.
 
== Frequently Asked Questions ==
 
= How do I use this plugin? =
 
You can remove Google Fonts and Google APIs from your website (optionally) and include web fonts which are hosted on your server (no third party content).

= Is this Plugin in compliance with GDPR =

Yes, it is intended to include web fonts on your server (no third party content like Google Fonts) and it gives you the opportunity to remove your existing (non GDPR compliant) Google Fonts and Google APIs from your Theme (if it might have any). 

= Can this Plugin cause problems to my Theme =

It depends on your Theme. If you decide to remove Google Fonts and Google Apis (non GDPR compliant third party code) from your Theme, but your Themeâ€™s functionality depends on those Google APIs code, it will break its functionality.
 
= How to uninstall the plugin? =
 
Simply deactivate and delete the plugin.
